Area

Description

The <area> tag defines an area inside an image map (an image map is an image with clickable areas).

<area> elements are always nested inside a <map> tag.

Example

Code Example:

          
            <p>Click on the sun or on one of the planets to watch it closer:</p>
            <img src="https://www.w3schools.com/tags/planets.gif" width="145" height="126" alt="Planets"
            usemap="#planetmap">
            <map name="planetmap">
                <area shape="rect" coords="0,0,82,126" href="https://www.w3schools.com/images/sun.gif" alt="Sun">
                <area shape="circle" coords="90,58,3" href="https://www.w3schools.com/tags/merglobe.gif" alt="Mercury"">
                <area shape="circle" coords="124,58,8" href="https://www.w3schools.com/images/venglobe.gif" alt="Venus">
            </map>
