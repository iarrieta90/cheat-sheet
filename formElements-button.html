Button Form Attribute

Description

The form attribute specifies the form the button belongs to.

The The value of this attribute must be equal to the id attribute of a <form> element in the same document.

Example

Code Example

                            <html>

                            <body>

                                <form action="/action_page.php" method="get" id="nameform">

                                    <label for="fname">First name:</label>

                                    <input type="text" id="fname" name="fname"><br>

                                    <label for="lname">Last name:</label>

                                    <input type="text" id="lname" name="lname">

                                </form>

                                <p>The button below is outside the form element, but still part of the form.</p>

                                <button type="submit" form="nameform" value="Submit">Submit</form>

                            </body>

                            </html>
