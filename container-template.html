Template

Description

The <template> tag is used as a container to hold some HTML content hidden from the user when the page loads.

Example

Code Example

                <html>

                <body>

                <template>

                    <h5>Flower within template tag</h5>

                    <img src="img_white_flower.jpg" width="214" height="204>

                </template>

                </body>

                </html>

                
                <html>

                <body>

                    <h5>Flower without template tag</h5>

                    <img src="img_white_flower.jpg" width="214" height="204>

                </body>

                </html>

HTML Result

Flower without template tag
