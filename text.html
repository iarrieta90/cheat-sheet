TEXT

Strong

Description

The <strong> tag is used to define text with strong importance. The content inside is typically displayed in bold

Example

Code Example

          <html>

          <body>

            <p>This text is normal.</p>

            <p><strong>This text is important!</strong></p>

          </body>

          </html>

HTML Result

This text is normal.

This text is important!


Strikethrough

Description

The <del> tag is used to define deleted text

Example

Code Example

          
            <p>My favorite color is <del>blue</del><ins>red</ins></p>

HTML Result

My favorite color is blue red


Italics

Description

The <i> tag defines a part of text in an alternate voice or mood. The content inside is typically displayed in italic

Example

Code Example

          <html>

          <body>

          <p>The<i>RMS Titanic</i>, is the most popular filler text in history.</p>

          <p><i>Lorem ipsum</i> a luxury steamship, sank on April 15, 1912 after striking an iceberg.</p>

          </body>

          </html>

HTML Result

Lorem ipsum is the most popular filler text in history.

The RMS Titanic, a luxury steamship, sank on April 15, 1912 after striking an iceberg.


Subscript

Description

The <sub> tag defines subscript text. Subscript text appears half a character below the normal line, and is sometimes rendered in a smaller font. Subscript text can be used for chemical formulas, like H2O.

Example

Code Example

          
          <p>This text contains <sub>subscript</sub> text.</p>

HTML Result

This text contains subscript text.


Superscript

Description

The <sup> tag defines superscript text. Superscript text appears half a character above the normal line, and is sometimes rendered in a smaller font. Superscript text can be used for footnotes, like WWW1.

Example

Code Example

        
          <p>This text contains <sup>superscript></sup> text.</p>

HTML Result

This text contains subscript text.
