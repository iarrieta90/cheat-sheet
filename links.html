Links

Description

The <a> tag defines a hyperlink, which is used to link from one page to another.

The most important attribute of the <a> element is the href attribute, which indicates the link's destination.

Example

Code Example:

                        <html>

                        <body>

                            <h5>Link being opened in a new browser tab</h5>

                            <a href="https://www.w3schools.com" target="_blank">Visit W3Schools.com!<a>

                            <p>If you set the target attribute to "_blank", the link will open in a new browser window or a new tab.<p>

                            <h5>Link being opened in the same browser tab</h5>

                            <a href="https://www.w3schools.com"  target="_parent">Visit W3Schools.com!<a>

                            <p>If you set the target attribute to "_parent", the link will open in the same window or a same tab.<p>

                            <h5>Link being refer to an existing ID of the HTML page</h5>

                            <a href="#linktitle">Go to the title of the page<a>

                        </body>

                        </html>

HTML Result:

Link being opened in a new browser tab
Visit W3Schools.com!

If you set the target attribute to "_blank", the link will open in a new browser window or a new tab.


Link being opened in the same browser tab
Visit W3Schools.com!

If you set the target attribute to "_parent", the link will open in the same window or a same tab.


Link being refer to an existing ID of the HTML page
Go to the title of the page
