Button

Description

The <input type="button"> defines a clickable button (mostly used with a JavaScript to activate a script).

Example

Code Example

                        
                    <p>The button below activates a JavaScript when it is clicked.</p>

                    <form>
                        <input type="button" value="Click me" onclick="msg()">
                    <form>
                            
                    <script>
                            function msg() {
                              alert("Hello world!");
                            }
                    </script>
