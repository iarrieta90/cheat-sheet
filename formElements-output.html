Output Form Element

Description

The <output> tag is used to represent the result of a calculation

Example

Code Example

                            <html>

                            <body>

                                <form oninput="x.value=parseInt(a.value)+parseInt(b.value)">

                                    <input type="range" id="a" value="50">

                                    +
                                    <input type="number" id="b" value="25">

                                    =
                                    <output name="x" for="a b">

                                </form>

                            </body>

                            </html>
